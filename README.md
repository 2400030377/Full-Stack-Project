
  # Bharat Samvidhan
  ## Indian Constitution Awareness Platform

  A modern React application designed to educate citizens about the Indian Constitution, its framers, fundamental rights, and the democratic principles that guide our nation.

  ### Features
  - Constitution Framers profiles with historical photographs
  - Interactive Preamble presentation
  - Constitutional History timeline
  - Multi-language support (14 Indian languages)
  - Professional government portal design
  - Responsive mobile-first layout

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
